## 基于bilibili清华大学微信小程序开发教程编写的 微信小程序 

#### App RN版本：<https://github.com/HuPingKang/LZKKDemo-RN>
#### b站清华大学小程序视频地址：<https://www.bilibili.com/video/av22004522>

#### 内容涉及：
     1.基础界面的搭建；
     2.网络请求、JSON Parse与数据渲染；
     3.音视频播放；
     4.地图：获取位置坐标、反地址解析、添加Markers；
     5.下拉刷新、上拉加载；
     6.其他；
